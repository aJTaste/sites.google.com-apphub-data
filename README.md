# Eaglercraft 1.12.2 WASM-GC版 データコピー（移行）の手順
中学卒業なので、もしも今のPCに保存されているEaglercraftを高校になってからも遊びたい人向けに、データの移行方法をまとめています。
中学卒業関係なくても、可能です。
友達とデータ共有できます。

## 移動後のEaglercraftでやること
- Eaglercraftを起動します。
- 起動が完了したら、Eaglercraftの画面から歯車アイコンを探し、クリックします。
- Consoleという項目が表示されたら、一番下の`>`の横の部分をクリックします。（テキスト入力状態になればOK）
- このリポジトリの`lsbakkuappu.js`の上の方のコードをコピーします。
- コピーしたコードを、二本指タップ（右クリック）から貼り付け、Excuteします。
`{"_eaglercraft_1.12.g":"dmVyc2lv...`
こんなのが出ればOK。
- 出力されたコードのブロックを選択し、コンソール右上のコピーアイコンからコピーし、移行後の環境からでも確認できる場所に保管します。
- ーーー
- このリポジトリの`kakidashi.js`のコードをコピーします。
- 先程と同様に、このコードをExcuteします。
- ファイルをダウンロードし、移行後の環境からでも確認できる場所に保管します。
 
 ## 移行後のEaglercraftでやること
 - `lsbakkuappu.js`の下の方のコードをコピーします。
 - コードの指示通りに、以前に保存したコード（1つ目の方）を該当部分に貼り付けます。
 - 同じ方法でExcuteします。
 - 指示通りに動作すれば、自動的にリロードし、設定やスキンなどのデータが移行されます。
 - ーーー
 - `yomikomi.js`のコードをコピーします。
 - 同様に、このコードをExcuteします。
 - 以前にダウンロードしたファイル（2つ目の方）をアップロードします。
 - 指示通りに動作すれば、自動的にリロードし、ワールドなどのデータが移行されます。
